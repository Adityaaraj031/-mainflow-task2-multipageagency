 NexaStudio – Responsive Multipage Agency Website

A modern and responsive multipage agency website built using HTML5, CSS3, and JavaScript. This project was developed as part of Mainflow Internship Task 2 and showcases a clean UI, responsive design, smooth animations, and JavaScript-based form validation.

✨ Features

* Responsive Design
* Multipage Layout
* Modern UI/UX
* Mobile Navigation Menu
* Contact Form Validation
* Smooth Hover Effects
* Cross-Browser Compatible

🛠️ Technologies

* HTML5
* CSS3
* JavaScript (ES6)
